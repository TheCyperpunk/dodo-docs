### Quick Integration Examples

<CodeGroup>
```typescript TypeScript
import { DodoPayments } from '@dodopayments/sdk';

const dodo = new DodoPayments({
  apiKey: process.env.DODO_API_KEY,
  environment: 'sandbox'
});

// Create a payment link
const paymentLink = await dodo.paymentLinks.create({
  productId: 'prod_123',
  amount: 2999,
  currency: 'USD'
});
```

```python Python
from dodopayments import DodoPayments

dodo = DodoPayments(api_key="your_api_key")

# Create a payment link
payment_link = dodo.payment_links.create(
    product_id="prod_123",
    amount=2999,
    currency="USD"
)
```

```bash cURL
curl -X POST 'https://api.dodopayments.com/v1/payment-links' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "product_id": "prod_123",
    "amount": 2999,
    "currency": "USD"
  }'
```
</CodeGroup>
